# dev-scala-cli-gradle-mysql-simple

## dev-Description
A scala gradle build, that connects to mysql database.

## dev-Tech stack
- scala
- gradle
  - 6.8.2

## dev-Docker stack
- mariadb:latest
- gradle:jdk11

## dev-To run
`sudo ./install.sh -u`

## dev-To stop
`sudo ./install.sh -d`

## dev-For help
`sudo ./install.sh -h`
